## Run Locally

Clone the project

```bash
  git clone [https://link-of-repo](https://github.com/Code-With-Amine/portfolio-amine-abaidi.git)
```

Go to the project directory

```bash
  cd portfolio-amine-abaidi
```

Install dependencies

```bash
  npm install or npm i
```

Start the server

```bash
  npm run dev
```
